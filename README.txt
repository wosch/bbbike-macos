BBBike
--------------

BBBike is a route-finder for cyclists in Berlin and Brandenburg.
A web interface is online: http://www.bbbike.de

The standalone program BBBike requires MacOS 10.5/Leopard
and X11 (standard part of MacOS).

To start BBBike, click on the wrapper script 

	./bbbike 

Other Cities around the World
-----------------------------

!!!Note: openstreetmap support is still in alpha test!!!
Data by openstreetmap.org

To start BBBike for your city, click on the city name:

	./Copenhagen
	./San_Francisco
	./Karlsruhe
